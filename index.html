Install aaPanel on Termux

Follow this detailed guide to easily install and set up aaPanel on your Termux environment for web hosting and management.

  • Prerequisites
  • Installation
  • Advanced Configurations
  • Troubleshooting

Prerequisites

Make sure you have the following installed before starting the installation:

  • Termux on your Android device
  • Ubuntu Root Filesystem (choose your desired version below):
    • Ubuntu Focal
    • Ubuntu Jammy
    • Ubuntu Bionic

Installation

Step 1: Install Termux and PRoot


pkg update && pkg upgrade
pkg install proot

Step 2: Download Ubuntu Root Filesystem


wget https://partner-images.canonical.com/core/focal/current/ubuntu-focal-core-cloudimg-arm64-root.tar.xz

Step 3: Extract and Set Up the Ubuntu Environment


mkdir ~/ubuntu-fs
tar -xf ubuntu-focal-core-cloudimg-arm64-root.tar.xz -C ~/ubuntu-fs

Step 4: Run Ubuntu via PRoot


proot -0 -w ~ -b /proc -b /sys -b /dev ~/ubuntu-fs/bin/bash

Step 5: Install aaPanel


bash <(curl -s http://www.aapanel.com/script/install-ubuntu.sh)

Advanced Configurations

After installation, configure the following services:

  • Web Server (Nginx/Apache)
  • Database Server (MySQL/PostgreSQL)
  • SSL Configuration

Troubleshooting

If you face any issues, try these solutions:

  • Ensure Termux is up-to-date: pkg update && pkg upgrade
  • Reinstall PRoot if necessary: pkg install proot
