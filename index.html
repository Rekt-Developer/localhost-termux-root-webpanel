Prerequisites

Before installing aaPanel, make sure you have the following installed:

  • Termux (for Android)
  • Ubuntu Root Filesystem (choose your desired version below):
    • Ubuntu Focal
    • Ubuntu Jammy
    • Ubuntu Bionic
    • Ubuntu Xenial

Installation

Step 1: Install Termux and PRoot


pkg update && pkg upgrade
pkg install proot

Step 2: Download Ubuntu Root Filesystem

Choose your desired version and download the root filesystem:


wget https://partner-images.canonical.com/core/focal/current/ubuntu-focal-core-cloudimg-arm64-root.tar.xz

Step 3: Extract and Set Up the Ubuntu Environment


mkdir ~/ubuntu-fs
tar -xf ubuntu-focal-core-cloudimg-arm64-root.tar.xz -C ~/ubuntu-fs

Step 4: Run Ubuntu via PRoot


proot -0 -w ~ -b /proc -b /sys -b /dev ~/ubuntu-fs/bin/bash

Step 5: Install aaPanel


bash <(curl -s http://www.aapanel.com/script/install-ubuntu.sh)

Advanced Configurations

After installation, you can configure the following services:

  • Web Server (Nginx/Apache)
  • Database Server (MySQL/PostgreSQL)
  • SSL Configuration

Troubleshooting

If you encounter issues, try the following solutions:

  • Ensure your Termux installation is up-to-date: pkg update && pkg upgrade
  • If `proot` is not working, try reinstalling it: pkg install proot

For more help, visit the GitHub Repository.
